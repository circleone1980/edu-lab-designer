## 7. 配对设计规则

### 主题配对原则

| 原则 | 说明 |
|------|------|
| 共享主题 | LARP 和推演围绕同一历史/现实事件 |
| 视角递进 | LARP 是"经历者"，推演是"决策者" |
| 时间错位 | LARP 是事件中/后，推演是事件前/中 |
| 数据连接 | LARP 数据自动传递到推演课 |

### 数量规范

- 4 对 LARP + 推演 = 8 个场景 + 2 个公共设计 + 1 个配对方案 = 13 文件
- 如用户要求更多/更少对，按比例增减

```
├── 00-方案源文档.md          (~500行) 原始材料/知识库
├── 01-PPT逐页大纲.md         (~120行) 16页骨架

```

## 3. 01-PPT逐页大纲模板

```markdown
# PPT 逐页大纲

## 元数据
- 总页数：16页
- 目标受众：[校领导/院长/投资人]
- 演讲目标：[获得立项批准/预算审批]
- 演讲策略：[从"为什么做"到"怎么做"到"多少钱"]
- 叙事弧线：为什么做→怎么组织→3个方向→技术支撑→可行性→成果

## 逐页骨架

P1: **封面** — [学校][学科]数智实验室建设方案
P2: **为什么现在做** — 政策窗口 × 技术成熟 × 学校契机
P3: **需求判断** — 3个痛点 + 核心定位
P4: **组织逻辑** — Kolb模型 × LARP+推演 × AI驱动
P5: **方向一·场景1** — [LARP场景概览]
P6: **方向一·场景2** — [推演场景概览]
P7: **方向一·效果** — 配对课程数据流和教学效果
P8: **方向二·AI中台** — AI能力架构
P9: **方向二·AI角色** — AI在教学中的具体角色
P10: **方向三·平台** — 教学管理与数据平台
P11: **三方向综合** — 架构全景图
P12: **技术架构** — 三层架构 + Mermaid图
P13: **空间布局** — 实验室物理空间设计
P14: **实施路径** — 两阶段时间线
P15: **预算** — 6个建设包百分比
P16: **预期成果** — 教学/教研/品牌三维成果
```

## 4. 02-PPT逐页详细描述模板

每页包含 4 部分：

```markdown
## PX: [页面标题]

### 页面目标
[这页要说服听众什么？1-2句话]

### 页面文案
[可直接粘贴到PPT的文案内容]

### 视觉建议
- 布局：[中央+模块 / 时间轴 / 左右对比 / 分层架构 / 空间分区 / 比例图]
- 核心视觉元素：[必须出现的图/表/图标]
- 配色提示：[与整体风格一致的色彩使用]
- 效果图：[如需生成，标注引用编号]

### 演讲提示
[演讲者在此页的要点，40-70秒的内容]
```

### 6种布局模式

| 模式 | 适用页面 | 特点 |
|------|---------|------|
| 中央+模块 | P2、P3 | 核心信息居中，辅助信息环绕 |
| 时间轴 | P14 | 左右延伸的时间线 |
| 左右对比 | P5-P6、P8-P9 | 两种内容并列展示 |
| 分层架构 | P11、P12 | 自上而下的层级结构 |
| 空间分区 | P13 | 平面布局图 |
| 比例图 | P15 | 饼图/柱状图展示比例 |

## 6. 04-PPT生成提示词模板

三层提示词策略：

### 第一层：完整版

```
你是一个专业的教育项目PPT设计师。请根据以下方案源文档，生成一份16页的PPT。

[粘贴完整的00-方案源文档内容]

要求：
- 16页，叙事弧线：P1封面→P2为什么→P3需求→P4组织→P5-P7方向一→P8-P9方向二→P10方向三→P11综合→P12架构→P13空间→P14实施→P15预算→P16成果
- 每页包含：标题、3-5个要点、1个核心视觉建议
- 风格：[来自03的基调]
- 受众：[目标受众]
```

### 第二层：视觉优化版

```
请优化以下PPT的视觉设计：

[粘贴01-逐页大纲]

优化方向：
- 每页的视觉布局建议（6种布局模式选一）
- 图表类型建议
- 效果图需求清单
```

### 第三层：格式优化版

```
请将以下PPT内容格式化为标准演示文稿格式：

[粘贴02-逐页详细描述]

格式要求：
- 每页独立区块
- 标题用H2
- 要点用无序列表
```

# 建设方案交付物模板

> ② 建设方案（7 文件）的完整模板和生成规范。

---

## 1. 目录结构

```
②-建设方案/
├── 08-整体建设详细方案.md    (~600行) 内部策略版，13章
├── 09-报价方案.md            (~270行) 6个建设包+百分比预算
├── 10-内部对照底稿.md        内部执行对照用
├── 11-建设方案完整版.md      (~580行) 客户端，11章
├── 12-效果图脚本与提示词.md  (~220行) Gemini生图剧本
├── 13-建设方案正式版.md      最终交付版（整合11+12+场景设计）
└── 14-执行计划.md            (~660行) 多文件协调的任务追踪
```

### 每章"五维价值"模板

每章结尾增加：

```markdown

| 维度 | 价值描述 |
|------|---------|
| 教学价值 | [本章建设内容如何提升教学质量] |
| 教研价值 | [如何支撑教师的教学研究] |
| 品牌价值 | [如何提升学校/学院的品牌形象] |
| 区域辐射 | [如何形成区域示范效应] |
| 长期资产 | [如何沉淀为可持续的数字资产] |
```

## 3. 09-报价方案模板

### 6个建设包

```markdown
# [学校][学科]数智实验室 · 报价方案

## 预算结构

| 建设包 | 内容 | 占比 | 预估金额 |
|--------|------|------|---------|
| A. 整体设计与项目管理 | 方案设计+项目管理+验收 | 6-8% | [金额] |
| B. 教室一：沉浸教学空间 | 硬件+装修+集成 | 18-22% | [金额] |
| C. AI 核心能力 | 大模型+RAG+知识图谱+Agent | 24-28% | [金额] |
| D. 课程内容与教室二 | 剧本+推演+教学设计+第二教室 | 28-32% | [金额] |
| E. 服务器与基础设施 | 计算+存储+网络+安全 | 6-8% | [金额] |
| F. 培训与运维 | 教师培训+技术培训+运维 | 6-8% | [金额] |
| **合计** | | **100%** | **[总金额]** |
```

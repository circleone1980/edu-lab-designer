# 通用学科适配指南

将 Kolb 体验式学习 + LARP/推演方法论适配到任意学科的核心指南。

---

## 1. 四维度分析框架

对任意学科，首先分析以下四个维度：

### 维度 1：体验场景类型

学科知识在什么物理/虚拟场景中"被体验"？

| 场景类型 | 典型学科 | 场景特征 |
|---------|---------|---------|
| 临床 | 医学、护理 | 医院、病房、手术室 |
| 法庭 | 法学 | 法院、审判庭、调解室 |
| 田野 | 农学、地质、考古 | 农村田间、矿山、遗址 |
| 工厂 | 工学、制造 | 生产线、工地、实验室 |
| 社区 | 社会学、教育学 | 社区、学校、村庄 |
| 商业 | 商学、管理学 | 会议室、市场、谈判桌 |
| 实验室 | 理学、药学 | 研究实验室、检测中心 |
| 历史 | 历史、马克思主义理论 | 历史现场（时间穿越） |
| 创作 | 艺术学、文学 | 工作室、舞台、编辑部 |

**适配步骤**：
1. 确定学科的 1-2 个核心场景类型
2. 该场景中有哪些"角色"（利益相关方）
3. 该场景中有哪些典型的"两难抉择"

### 维度 2：核心冲突类型

该学科最常面对的价值/伦理冲突是什么？

| 冲突类型 | 描述 | 典型学科 |
|---------|------|---------|
| 伦理 | 个人道德 vs 专业职责 | 医学、法学 |
| 效率 | 速度 vs 质量 | 工学、管理学 |
| 公平 | 资源分配正义 | 社会学、经济学 |
| 安全 | 风险 vs 收益 | 工学、医学 |
| 创新 | 传统 vs 突破 | 艺术、理学 |
| 传承 | 保护 vs 发展 | 文化遗产、农学 |
| 信任 | 透明 vs 隐私 | 新闻学、法学 |
| 忠诚 | 个人 vs 集体 | 军事、管理学 |

**适配步骤**：
1. 列出该学科的 2-3 个核心冲突类型
2. 每种冲突设计一个"极端情境"（让冲突不可回避）
3. 确认该冲突有明确的"正反方"（可辩论）

### 维度 3：知识形态

该学科的知识以什么形态存在？

| 形态 | 描述 | LARP 中如何呈现 | 推演中如何呈现 |
|------|------|---------------|--------------|
| 理论 | 概念、原理、模型 | 角色在对话中自然引用 | 论证陈述中引用 |
| 技能 | 操作、方法、流程 | 角色必须执行某操作 | 分析具体操作步骤 |
| 态度 | 价值观、立场、信念 | 角色做出情感选择 | 学生表明立场并论证 |
| 判断 | 综合、权衡、决策 | 角色面对两难选择 | 学生独立做决策 |
| 创造 | 设计、方案、作品 | 角色提出创新方案 | 学生设计替代方案 |

### 维度 4：价值教育方向

该学科需要培养哪些价值/态度？

| 方向 | 描述 | 适用学科 |
|------|------|---------|
| 职业伦理 | 专业道德规范 | 医学、法学、工学 |
| 社会责任 | 对社会/公众的义务 | 所有学科 |
| 家国情怀 | 国家认同、民族自豪 | 马克思主义理论、历史 |
| 创新精神 | 突破、探索、批判思维 | 理学、工学、艺术 |
| 人文关怀 | 同理心、尊重、包容 | 医学、教育学、艺术 |
| 团结协作 | 集体意识、合作精神 | 工学、农学、军事 |
| 生态文明 | 环保、可持续 | 农学、工学、理学 |

## 2. LARP 主题发现方法论

为任意学科发现适合的 LARP 主题：

### 步骤 1：列出重大争议/转折事件

使用 Web 搜索：`[学科] 历史 重大事件 争议`、`[学科] 伦理 案例 争议`

筛选标准：
- 有明确的"冲突双方"
- 有真实的"历史后果"
- 可在 70 分钟内呈现完整弧线
- 学生可以产生情感共鸣

### 步骤 2：筛选有"冲突+选择"的事件

必须满足：
- 至少一方有多个可选行动
- 每个选择都有合理理由
- 选择结果对当事人有重大影响
- 存在"道德灰色地带"（非简单黑白）

### 步骤 3：确认可转化为 5 幕结构

| 幕次 | 对应 | 内容要求 |
|------|------|---------|
| 第0步 | 准备 | 能否设计开场动画和角色卡？ |
| 第1幕 | 引入 | 能否在 15 分钟内建立角色立场？ |
| 第2幕 | 冲突 | 冲突能否升级到不可回避？ |
| 第3幕 | 抉择 | 是否有 3 个可选行动？ |
| 第4幕 | 信 | 能否连接过去与现在？ |
| 第5幕 | 复盘 | 能否映射到学科理论？ |

### 步骤 4：检查本地/区域连接

搜索：`[地区] [学科相关事件]`、`[学校名] [学科] 历史`

优先选择：
- 发生在本地区的事件
- 本校师生参与过的事件
- 与地方文化特色相关的事件

## 3. 推演主题发现方法论

### 步骤 1：从配对 LARP 提取核心两难

每个 LARP 都有一个核心冲突。推演将这个冲突从"角色视角"升级为"决策者视角"。

**升级公式**：
- LARP 角色是"受影响者" → 推演中学生在"影响者/决策者"位置
- LARP 中的选择是"个人选择" → 推演中的选择是"公共政策/集体决策"

### 步骤 2：设计决策框架

确定推演的核心问题，格式：
"[决策者身份]面对[两难情境]，应该选择[A]还是[B]？"

示例：
- 医学："院长面对有限呼吸机，应该优先给年轻患者还是重症患者？"
- 法学："法官面对疑罪案件，应该坚持疑罪从无还是考虑社会安全？"
- 工学："总工程师面对安全隐患报告，应该停工整改还是赶工期？"

### 步骤 3：设计三视角资料包

为每个选项准备对立证据：

| 视角 | 内容 | 来源模拟 |
|------|------|---------|
| A 方 | 支持选项 A 的数据和论证 | 学科文献、统计数据、案例 |
| B 方 | 支持选项 B 的数据和论证 | 学科文献、统计数据、案例 |
| 第三方 | 旁观者/后世/跨文化视角 | 历史记录、比较研究、伦理分析 |

每视角 4-5 条材料，每条 50-100 字。

## 4. 角色原型推导方法论

### 步骤 1：绘制利益相关方矩阵

分析该场景中所有受影响的群体：

```
                高影响
                  |
     决策者 ────┼──── 直接受影响者
                  |
                低影响
         间接影响者 ──── 观察者
```

### 步骤 2：选取 5-6 个代表性视角

从矩阵中选取覆盖不同立场的角色：

| 角色位置 | 选取原则 | 数量 |
|---------|---------|------|
| 决策者 | 推演课视角，不在 LARP 中 | 0-1 |
| 直接受影响者（支持） | 选择会使其受益 | 1-2 |
| 直接受影响者（反对） | 选择会使其受损 | 1-2 |
| 中间派 | 立场摇摆，有内在冲突 | 1 |
| 意外视角 | 跨阶层/跨文化 | 0-1 |

### 步骤 3：角色差异化设计

确保 5 个角色覆盖：

| 差异维度 | 设计要求 |
|---------|---------|
| 阶层 | 至少覆盖 2 个社会阶层 |
| 性别 | 至少 1 名女性 |
| 年龄 | 覆盖不同年龄段 |
| 民族（如适用） | 至少 1 个少数民族 |
| 立场 | 2 支持、2 反对、1 中间 |
| 秘密 | 每人有推动剧情的私密信息 |

## 6. 适配验证清单

完成学科适配后，用此清单验证：

- [ ] 确定了 1-2 个体验场景类型
- [ ] 确定了 2-3 个核心冲突类型
- [ ] 确定了主要知识形态（理论/技能/态度/判断）
- [ ] 确定了 2-3 个价值教育方向
- [ ] 发现了 3-4 个 LARP 主题（含本地连接）
- [ ] 每个 LARP 主题可转化为 5 幕结构
- [ ] 发现了 3-4 个配对推演主题
- [ ] 每个推演有明确的"A vs B"核心问题
- [ ] 设计了 5-6 个差异化角色原型
- [ ] 角色覆盖不同阶层/立场/性别
- [ ] 三视角资料包的立场分布合理

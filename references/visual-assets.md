# 视觉素材生成指南

> 为每个 LARP 剧本和推演脚本生成配套的视觉素材。
> 集成 `gemini-image-gen` Skill 和 `gemini-cn-prompt-patterns` Skill。

---

## 1. 素材类型清单

| 素材类型 | 数量/剧本 | 格式 | 生成方式 |
|---------|----------|------|---------|
| AR 沉浸场景图 | 5张（每幕1张） | 16:9 PNG | gemini-image-gen |
| 角色肖像 | 5-6张 | 肖像 PNG | gemini-image-gen |
| 关键线索/道具图 | 3-5张 | PNG | gemini-image-gen |
| 推演场景概念图 | 1-2张 | 16:9 PNG | gemini-image-gen |

---

## 2. 统一风格总控词

每个项目的所有图片应共享统一的视觉风格：

```
[统一风格总控词模板]

视觉风格：[水墨/写实/手绘/赛博朋克]风格
色调：[暖黄/冷蓝/黑白+局部彩色]
时代感：[具体时代]的中国[地区]
构图特点：[居中/三分法/对称]
细节要求：历史考究，服饰/建筑/道具符合时代
文字元素：使用[书法字体/印刷体]呈现标题
禁止元素：现代物品、品牌标识、版权素材
输出规格：[分辨率] PNG，[宽高比]
```

### 各学科风格建议

| 学科 | 推荐风格 | 色调 |
|------|---------|------|
| 马克思主义理论 | 历史水墨/写实 | 暖黄+红 |
| 医学 | 临床写实 | 冷白+蓝 |
| 法学 | 正式庄重 | 深色+金 |
| 工学 | 技术蓝图 | 蓝灰+白 |
| 农学 | 自然手绘 | 绿+暖黄 |
| 商学 | 现代简约 | 灰+蓝 |
| 教育学 | 温暖手绘 | 暖色系 |
| 理学 | 科学可视化 | 冷色系 |
| 艺术学 | 风格化 | 按时代 |

## 3. AR 沉浸场景图（每幕1张）

### 5张图的叙事弧线

| 幕次 | 场景图主题 | 情绪基调 | 光线 |
|------|----------|---------|------|
| 第1幕 | [初始场景] | 紧张/期待 | 暗调/渐变 |
| 第2幕 | [冲突场景] | 升级/压迫 | 对比强烈 |
| 第3幕 | [高潮场景] | 紧迫/抉择 | 聚光灯效果 |
| 第4幕 | [时空连接] | 温暖/怀念 | 柔和光线 |
| 第5幕 | [反思/现代] | 平静/启示 | 明亮自然光 |

### 提示词模板（Type A：效果图）

```
[统一风格总控词]

场景：[具体场景描述]
时代：[时间]，[地点]
画面内容：[详细描述画面中的所有元素]
人物：[人物数量/位置/动作描述，不含具体面孔]
氛围：[情绪关键词]
关键视觉元素：[必须包含的标志性元素]

构图：[具体构图要求]
视角：[俯视/平视/仰视]
焦点：[画面中心焦点]

禁止：现代物品、文字水印、品牌标识
输出：1920×1080 PNG，16:9
```

## 4. 角色肖像（5-6张）

### 肖像规范

| 要素 | 要求 |
|------|------|
| 构图 | 半身像，面部清晰 |
| 服饰 | 符合时代和身份 |
| 背景 | 虚化/纯色，不喧宾夺主 |
| 表情 | 体现角色性格和内心状态 |
| 尺寸 | 至少 512×512 |

### 提示词模板（Type B：角色图）

```
[统一风格总控词]

角色肖像：[角色名]
身份：[身份描述]
年龄：[年龄]岁
民族：[民族]
时代：[时间]的[地区]
服饰：[具体服饰描述]
表情：[表情描述，体现角色内心]
道具：[手持或身边的标志性物品]

背景：虚化处理，暗示[角色所处的环境]
光线：[光线方向和质感]

禁止：现代妆容、时尚元素
输出：[分辨率] PNG，肖像比例
```

## 5. 关键线索/道具图（3-5张）

### 线索图类型

禁止：现代物品混入
输出：[分辨率] PNG

## 7. 批量生成流程

### 出图顺序

```
1. 统一风格总控词 → 确认风格一致性
2. AR场景图 ×5（按幕次顺序）
3. 角色肖像 ×5-6（可与场景图并行）
4. 关键线索图 ×3-5（最后生成）
5. 推演概念图 ×1-2（与推演脚本配合）
```

### 修正命令

如果某张图不满意，使用修正命令：

```
[原始提示词]

修正要求：
- [具体修改点1]
- [具体修改点2]
保持其他元素不变。
```

### 质量检查

- [ ] 所有图片风格统一
- [ ] 历史考究（服饰/建筑/道具符合时代）
- [ ] 无现代物品混入
- [ ] 人物肖像表情与角色性格一致
- [ ] 场景图情绪与对应幕次匹配
- [ ] 图片分辨率达标
- [ ] 无水印/品牌标识

# 网络调研策略

> Phase 1 调研发现阶段的搜索策略和方法论。

---

## 1. 调研维度（6个）

| 维度 | 目标 | 搜索方向 |
|------|------|---------|
| 学校背景 | 办学特色、校训、优势学科 | 学校官网、百科 |
| 区域历史 | 重大事件、重要人物、文化特色 | 地方志、百科 |
| 学科本地连接 | 该学科在本地区的独特故事 | 地方+学科交叉搜索 |
| 校友/名人 | 该校该学科走出的杰出人物 | 校史、百科 |
| 教学痛点 | 该学科教学的共性问题 | 学术论文、教育论坛 |
| 政策机遇 | 教育部/省级相关政策 | 政府网站、新闻 |

---

## 2. 搜索策略

### 三轮搜索法

#### 第一轮：宏观扫描

获取学校、地区、学科的大图景：

```
搜索模板：
- "[学校名] 发展历程 特色"
- "[地区] 历史 文化 特色"
- "[学科] 教学 痛点 改革"
```

目标：建立整体认知框架，识别值得深入的线索。

#### 第二轮：深度挖掘

针对第一轮发现的有价值线索深入搜索：

```
搜索模板：
- "[学校名] [学科/学院] 发展历程"
- "[地区] [学科关键词] 历史事件 人物"
- "[学校名] 杰出校友 [学科]"
- "[学科] 伦理争议 案例 2020-2025"
```

目标：获取具体的人物、事件、数据，可作为剧本素材。

#### 第三轮：交叉验证

验证第二轮发现的素材的准确性：

```
搜索模板：
- "[事件名] 真相 史料"
- "[人物名] 生平 贡献"
- "[地区] [学科关键词] 故事 案例"
```

目标：确保素材的真实性和可靠性。

## 3. 搜索查询生成

### 动态生成模板

不使用固定搜索列表，而是根据输入动态生成：

| 维度 | 搜索模板 | 填充变量 |
|------|---------|---------|
| 学校 | `[学校名] [学科/学院] 发展历程 特色` | 学校名、学科 |
| 区域 | `[地区] [学科关键词] 历史事件 人物` | 地区、学科关键词 |
| 学科 | `[学科] 伦理争议 案例 2020-2025` | 学科 |
| 人物 | `[学校名] 杰出校友 [学科]` | 学校名、学科 |

| 政策 | `[学科] 教育部 虚拟仿真 政策 2024-2025` | 学科 |
| 地方连接 | `[地区] [学科关键词] 故事 案例` | 地区、学科关键词 |

## 4. 素材评估标准

### 用于 LARP 的素材

| 标准 | 说明 |
|------|------|
| 冲突性 | 有明确的对立双方 |
| 不可回避性 | 冲突必须被面对，无法回避 |
| 道德灰色地带 | 非简单黑白，有合理理由支持不同选择 |
| 情感共鸣 | 学生能产生代入感 |
| 时长适配 | 可在 70 分钟内完整呈现 |

### 用于推演的素材

| 标准 | 说明 |
|------|------|
| 决策性 | 有明确的决策者角色 |
| 信息不对称 | 支持不同选择的证据各有道理 |
| 后果可推演 | 不同选择的后果可以合理推演 |
| 真相可验证 | 有历史/事实真相可供揭示 |
| 理论映射 | 可映射到学科核心理论 |

### 1. 学校与学院概况

- 校训与教育理念

- 故事3：[标题]（[时间段]）— 简述

### 4. 教学痛点分析
- 痛点1：[描述]
- 痛点2：[描述]
- 痛点3：[描述]

- [政策1]：[简述及对本项目的意义]

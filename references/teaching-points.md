# 教学点提取方法论

> 从 LARP 剧本和推演脚本中系统提取教学点的方法论。

---

## 1. 双线提取框架

每个剧本/推演完成后，需要提取两条线的教学点：

### 专业知识线

本学科核心知识点，每剧本至少 3 个。

### 价值教育线

思政/伦理/职业素养/社会责任等维度，每剧本至少 3 个维度。

---

## 2. 提取流程

### 步骤 1：标记关键情节

通读剧本/推演脚本，标记所有"学生必须做出选择/判断"的情节节点。

| 节点类型 | 标记方式 | 示例 |
|---------|---------|------|
| 事实判断 | 学生需要判断某信息的真假 | "红军真的不扰民吗？" |
| 价值选择 | 学生需要在两种价值间取舍 | "开城 vs 死守" |
| 情感共鸣 | 学生产生强烈情感反应 | "六七百人参军" |
| 理论关联 | 可直接映射到学科理论 | "民心向背" |

### 步骤 2：映射到理论

对每个关键情节节点，寻找对应的学科理论：

```
| 场景中的情节 | 对应理论/知识 | 理论解析 |
|-------------|-------------|---------|
| [具体情节]   | [理论概念]   | [为何体现此理论] |
```

**映射原则**：
- 理论必须是本学科课程大纲中的知识点
- 解析要说明"为什么这个情节体现了这个理论"
- 避免牵强附会——如果情节与理论关联度低于 70%，不纳入

### 步骤 3：设计价值维度

确定每个情节涉及的价值教育维度：

| 维度 | 判断标准 | 适用场景 |
|------|---------|---------|
| 职业伦理 | 涉及专业行为准则 | 医学、法学、工学 |
| 社会责任 | 涉及对公众/社会的义务 | 所有学科 |
| 家国情怀 | 涉及国家认同、民族自豪 | 历史、马克思主义理论 |
| 创新精神 | 涉及突破、探索、批判思维 | 理学、工学、艺术 |
| 人文关怀 | 涉及同理心、尊重、包容 | 医学、教育学、艺术 |
| 团结协作 | 涉及集体意识、合作精神 | 工学、农学 |
| 生态文明 | 涉及环保、可持续发展 | 农学、工学、理学 |

---

## 3. 理论映射表格式

### LARP 剧本理论映射表

```
| 游戏中的情节 | 对应理论/知识 | 理论解析 |
|-------------|-------------|---------|
| [情节1] | [理论1] | [解析] |
| [情节2] | [理论2] | [解析] |
| ... | ... | ... |
```

要求：
- 至少 6 行映射（专业知识3+价值教育3）
- 每行解析 50-100 字
- 理论名称使用学科标准术语

### 推演脚本理论映射表

```
| 推演中的抉择 | 对应理论 | 理论解析 |
|-------------|---------|---------|
| [抉择1] | [理论1] | [解析] |
| [抉择2] | [理论2] | [解析] |
| ... | ... | ... |
```

要求同上。

## 4. AI 个性化反思提示设计

### 设计原则

- **不评判**：不说"你的选择是对/错的"
- **不引导**：不暗示"正确答案"
- **只提问**：用问题引发思考
- **个性化**：基于学生实际行为生成

### 提示模板

#### LARP 课后反思提示

```
[根据学生角色和选择生成]

你在《[剧本名]》中扮演了[角色名]。
在[关键时刻]，你选择了[选择内容]。

请思考：
- 你为什么做了这个选择？如果你是[另一个角色]，你会做同样的选择吗？
- [角色名]的[某个行为]，你觉得是[价值A]还是[价值B]的体现？
- 一周后的推演课上，你将以自己的身份面对类似的选择。你觉得你会怎么选？
```

#### 推演课后反思提示

```
[根据学生投票和陈述生成]

你在推演中选择了[XX]，主要理由是[从陈述中提取]。

请思考：
- 你的选择和真实历史一致吗？如果不一致，差异在哪里？
- 听了其他同学的理由后，你的想法有变化吗？
- 上一堂 LARP 课你扮演[角色名]时的经历，如何影响了今天的判断？
- [学科理论]如何解释你今天的选择？
```

### 间隔消化期推送内容

| 时间点 | 推送内容 |
|--------|---------|
| LARP课后第1天 | 回顾角色经历，提示反思 |
| LARP课后第3天 | 推送1-2条相关拓展资料 |
| LARP课后第5天 | 预告推演课视角切换 |

## 5. 教学数据收集

### 自动收集的数据

| 数据类型 | LARP 课 | 推演课 |
|---------|--------|-------|
| 选择数据 | 各幕选择、最终抉择 | 投票选择、修正轨迹 |
| 行为数据 | 线索查看顺序、讨论发言次数 | 资料查阅顺序、查阅时长 |
| 情感数据 | 情绪变化（AI 分析） | 立场转变轨迹 |
| 文本数据 | 角色内心独白、信件 | 陈述理由、反思日记 |

### 数据应用

- **教学分析报告**：每个学生的参与度、选择倾向、情感触点
- **对比分析**：同一学生在"角色身份"和"真实身份"下的选择差异
- **课程改进**：基于数据调整后续课程设计
- **学术研究**：支撑教改论文的原始数据
